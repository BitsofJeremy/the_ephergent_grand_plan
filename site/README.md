# ephergent.com — Phase 07 Site

## Overview
This folder contains the technical and content scaffolding for the ephergent.com site, as specified in the GRAND_MASTER_PLAN and content_map.md.

- Astro 5 + Tailwind 3 (static site generation)
- Godot 4 HTML5 games (browser-playable, 15MB cap)
- Content collections: games, crew, lore, transmissions
- Lore Atlas: not a wiki, entries unlock as story/games progress
- No nautical metaphors, sci-fi vocabulary only
- Transmission schema: title, author, date, tags, excerpt

See astro_scaffold.md for directory and content structure.
