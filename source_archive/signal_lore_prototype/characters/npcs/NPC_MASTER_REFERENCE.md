## Regional Naming Conventions Verified

- **Cuprum**: Guild titles + working names (Guildmaster, Master, guild-specific titles). Ratchet and Anvil are working names. Freeboters use first name + surname (Anvil Voss).
- **Nocturnis**: Elder/Resonant rank + single name. Tonal names for natives (spoken + sung components). Siran and Kael are shortened for outsider use.
- **Verdanya**: Nature/growth titles + single name. Thessa, Lira. Branch-Mother is a rank.
- **Shattered Span**: Earned names (Icecutter, etc.) + birth names. Scrimshaw records achievements.
- **Nexus**: Scholar rank + single name. Scholar-Prime is highest rank. Marin follows academic convention. Underbright workers use first name + district name (Tally Meridian).
- **The Quiet**: Regional title + name (Forge-Sister, Branch-Mother) or standalone (Maren as leader).
