# Story Generation Prompts

This directory contains prompts and system instructions for AI-assisted story generation.

## Files

- `EPISODE_GENERATION_PROMPT.md` - Complete writing guidelines for episodes
- `RADIO_PLAY_CONVERSION_PROMPT.md` - Instructions for converting prose to radio play format
- `ephergent_universe_system_prompt.md` - System-level worldbuilding context and rules

## Usage

These prompts are used with Claude or other LLMs to:
1. Generate new episode prose
2. Convert prose episodes to radio play scripts
3. Maintain consistency across the Ephergent Universe

All prompts emphasize character voice, worldbuilding rules, and the brand's tone.
