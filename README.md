# The Ephergent — Merged Universe (Green Field)

> **Company**: Before Greatness LLC  
> **Product**: Browser-playable games + transmissions + comics, all on ephergent.com  
> **Status**: Full overhaul — three creative visions to choose from

## What We're Building Toward
Free, browser-playable Godot games (15MB cap). Each game is a different genre, a different crew member's lens on the same universe. The story IS the game delivery mechanism.

## The Synthesis Rules
- Dimensions are **frequencies** on a cosmic dial — never change this
- **Space**, not Sea. Sci-fi vocabulary only.
- **A1** = ancient quantum navigator → espresso machine → ship. Sacred.
- **Clive** = ancient Builder Companion Bot (signal_lore form: knee-high robot, glowing sphere head, fedora, noir voice). Barry backstory kept and adapted.
- **Mochi** = simple Builder artifact. Warm, dome-shaped, pulses near Builder tech. No consciousness arc.
- **The Builders** = mystery civilization. Built the frequency infrastructure. Vanished 800 years ago in The Dimming.
- **The Drift** = ongoing entropy as Builder infrastructure fails. Not a villain.
- **No corporate villains**. Antagonists are philosophical, environmental, or human-scaled.
